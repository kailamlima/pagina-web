Gerador de Cartão de Visitas Digital

Crie cartões criativos, coloridos e um pouco exagerados para apresentar sua versão digital.

Projeto Gerador Dicas GitHub

Sobre o projeto

Este site é um exercício simples de HTML, CSS e JavaScript para gerar um cartão de visitas digital com nome, cargo exótico e visual personalizado.

A proposta é transformar uma ideia direta em uma página organizada, usando estrutura semântica, layout tableless e uma interação feita com JavaScript puro.

Gerar cartão

Clique no botão, informe um nome e veja o cartão aparecer com um cargo diferente.

Dicas do cartão

  • Use um nome curto para deixar o cartão mais elegante.
  • Cada clique gera uma combinação visual diferente.
  • O número do cartão acompanha a quantidade de gerações.

Componentes

Nome, cargo exótico, número do cartão e fundo colorido com efeito visual.
