# Gerador de Cartão de Visitas Digital

Projeto simples de faculdade desenvolvido para a disciplina Programação para Internet.

## Objetivo

Criar uma página HTML, CSS e JavaScript que gere cartões de visitas digitais com nome, cargo exótico, número do cartão e estilo visual aleatório.

## Tecnologias usadas

- HTML5
- CSS3
- JavaScript

## Como abrir o site

Abra o arquivo `index.html` em qualquer navegador moderno.
